Git
Primeiro organizar as branches na minha propria maquina

Para enviar as mudancas ao repositorio do github eh:
git push -u origin master

Stage changes:
git add .

Commit
git commit -m "texto do commit. Isso eh obrigatorio"

Deletar branches remotos
git push origin --delete test
git branch -d test
(Test eh o nome da branch ne)

git stash
git reset --hard [hashcode]
git rebase
git fetch

git status
Checa para ver como esta o seu codigo, se deu commit, add e outras coisas
